# vim_installer

---

#### 1. 说明

一个简易的常用 VIM 及插件的快速安装库。


#### 2. 安装

分步安装
1. `git clone https://github.com/XingangShi/vim_installer.git`
2. `cd vim_installer`
3. `bash setup.sh`

一键安装

```python
git clone https://github.com/XingangShi/vim_installer.git && cd vim_installer && bash setup.sh
```
